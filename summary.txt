Battery Pack Designer - Project Summary
📦 What It Is
Battery Pack Designer is a standalone package extracted from the main ThinkClock BatteryScope project that focuses exclusively on one feature: designing optimal battery packs from a library of characterized battery cells.

Created on March 26, 2026, it's a self-contained application split into:

Frontend: React/Vite + MUI component library (13 React components)
Backend: Python Flask API with smart cell distribution algorithm
Location: battery-pack-designer folder within the main project
🎯 What It Does
Core Functionality
Designs optimal battery packs by:

Fetches Cell Inventory from VPS storage (/optv1 container)

Pulls characterized cells from training_daq_* or prediction_daq_* files
Enriches with training_metadata.json (SOH, capacity predictions)
Only includes cells with valid SOH and capacity data
Smart Cell Selection Algorithm

Calculates composite score for each cell:
Capacity (40% weight)
SOH/State of Health (40% weight)
Internal Resistance/IR (20% weight, inverse)
Groups cells by score (bins of 5 points)
Sorts cells from best to worst
Optimal Pack Distribution

Distributes cells using round-robin algorithm
Balances parallel strings to avoid weak links
Fills pack in series/parallel configuration (e.g., 7S22P)
Handles partial filling if insufficient cells available
Visualization & Analytics

Pack Matrix View: Visual grid showing cell placement
Pack Composition: Statistical breakdown per parallel string
Inventory Distribution: Histogram of capacity ranges
Cell Performance Plots: Individual and master plots
Warnings: Alerts for capacity/IR mismatches
